To Get The Most Out of This Course:

Review these steps before beginning this course:

  • CHECK to make sure your screen resolution is set to 1024x768.
  • REMEMBER, as you move through the course, to either PRINT or SAVE module summaries, job aids, and references when you see them. These contain useful information that will enhance your learning experience.
  • REVIEW THE HELP FILE, which provides the following information about the course:
    • Computer and Internet browser settings
    • Software requirements
    • How to move through the course modules
    • How to interact with the types of screens you will see in the course
  • WHEN EXITING THE COURSE, be sure to close all windows (the course, the course launch window, and TRAIN.org window).  This way, when you go back into the course, you will be on the last screen you were working on, and will be able to start again from that point. 

Note: The Help file is also accessible by clicking the Help button, located in the lower left corner of every screen.
